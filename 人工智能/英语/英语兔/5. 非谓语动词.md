## 非谓语动词核心概念 [原片 @ 00:00](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=0)
### 限定动词 (Finite Verbs) 的特征 [原片 @ 01:50](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=110)
1.  **三大功能**：表达动作的时间、状态、语气（通过动词变位实现）
    -   例：`The rabbit eats carrots`（第三人称单数现在时）
    -   例：`The rabbit ate a carrot`（过去时）
2.  **受主语限制**：必须与主语人称和数一致
    -   例：`The rabbit eats` (√) vs `The rabbit eat` (×)
3.  **句子核心**：一个简单句只能有一个谓语动词 [原片 @ 04:00](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=240)

### 非谓语动词 (Non-finite Verbs) 的特征 [原片 @ 02:58](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=178)
1.  **无限制性**：不受主语人称、数、时态限制
    -   例：`to eat`, `eating`, `minced` 在句中不随主语变化
2.  **失去核心功能**：无法独立表达动作的时间、状态、语气
3.  **四大类型**：动词不定式、动名词、现在分词、过去分词 [原片 @ 08:22](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=502)
4.  **定位**：只能出现在句子的非核心成分（主语、宾语、定语、状语等）[原片 @ 03:46](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=226)

## 动词不定式 (Infinitive) [原片 @ 05:54](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=354)
### 形式与命名 [原片 @ 05:58](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=358)
1.  **完全不定式 (Full Infinitive)**：`to` + 动词原形（如 `to eat`）
2.  **裸不定式 (Bare Infinitive)**：省略 `to`（如 `help me cook`）*![](http://localhost:8000/static/screenshots/screenshot_097b9b45b-f64d-474e-a9e5-71d0882722df.jpg)
3.  **词源**：`Infinitive` = 无限（不受时态人称限制）

### 句法功能与例句 [原片 @ 07:28](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=448)
| 成分       | 例句                                                                 | 功能说明                                                                 |
| :--------- | :------------------------------------------------------------------- | :----------------------------------------------------------------------- |
| **主语**   | `To eat a carrot daily is healthy.` → `It is healthy to eat...`      | 用 `it` 作形式主语避免头重脚轻 [原片 @ 09:32](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=572)                          |
| **宾语**   | `The rabbit likes to eat carrots.`                                   | 部分动词后需接不定式（如 `want`, `decide`）                              |
| **宾补**   | `The rabbit expected the wolf to eat a carrot.`                      | 补充说明宾语动作 [原片 @ 11:47](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=707) <br> `make/let` 后需用裸不定式 *![](http://localhost:8000/static/screenshots/screenshot_1e2bd4655-3b4b-4ca3-8803-fc3ee0edf8d2.jpg) |

## 动名词 (Gerund) [原片 @ 16:01](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=961)
### 本质与形式 [原片 @ 16:16](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=976)

## 现在分词 (Present Participle) [原片 @ 22:24](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=1344)

## 过去分词 (Past Participle) [原片 @ 28:19](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=1699)
### 形式与本质 [原片 @ 28:30](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=1710)
1.  **规则形式**：动词原形 + `-ed`（如 `minced`）
2.  **不规则形式**：需积累（如 `eaten`, `seen`, `cut`）[原片 @ 29:28](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=1768)
3.  **核心属性**：表**被动**或**完成**，**不带时间概念**（例：`minced carrot` 仅表"被切碎"）

## AI 总结
### 非谓语动词核心对比表
| **类型**     | **功能定位**          | **形式特征**       | **典型例句**                     |
| :----------- | :-------------------- | :----------------- | :------------------------------- |
| **不定式**   | 表目的/未来动作倾向   | `to do` / 裸不定式 | `He wants to eat.`               |
| **动名词**   | 名词性成分（动作本身）| `doing`            | `Eating is fun.`                 |
| **现在分词** | 主动进行/修饰主动者   | `doing`            | `The running rabbit.`            |
| **过去分词** | 被动完成/修饰受动者   | `done`             | `The stolen carrot.`             |

### 使用要点
1.  **一山不容二虎**：简单句唯一谓语动词，其余动词需转为非谓语形式 [原片 @ 04:38](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=278)
2.  **功能决定形式**：
    -   名词性成分 → 动名词/不定式
    -   形容词性修饰 → 现在/过去分词
3.  **主动 vs 被动**：
    -   主动关系 → 现在分词 (`interesting`)
    -   被动关系 → 过去分词 (`interested`)
4.  **简化从句**：非谓语可替代状语/定语从句（如 `to get carrots` = `in order to get carrots`）[原片 @ 14:45](https://www.bilibili.com/video/BV1XY411J7aG_p6?t=885)
