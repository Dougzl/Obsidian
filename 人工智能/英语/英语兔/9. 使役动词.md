## 使役动词 (Causative Verbs) 语法精讲 [原片 @ 00:00](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=0)

### 使役动词的定义与核心特征 [原片 @ 01:22](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=82)
*   **定义**：表达“使、让、令”等含义，要求宾语执行或经历某个动作的动词。
*   **核心结构**：`使役动词 + 宾语 (Object) + 宾语补足语 (Object Complement)`。
    *   宾语：动作的承受者 (如：the wolf)。
    *   宾语补足语：对宾语的补充说明（动作或状态），必不可少。*![](screenshots/screenshot_0fa512bcc-9629-4e6e-b9ee-1e13590ae724.jpg) (展示句子结构不完整)
*   **宾语补足语形式**：
    *   动词原形（裸不定式 Bare Infinitive）: `make/let/have + object + V原形` (e.g., eat, wash) *![](screenshots/screenshot_1491fe055-74a5-4eeb-813d-9fc76a92cfe0.jpg) (展示 make + object + V原形)
    *   形容词 (Adjective)：`make + object + Adj` (e.g., full) *![](screenshots/screenshot_205964942-ab29-4e94-9d93-6f52c01de230.jpg) (展示 made the wolf full)
    *   副词 (Adverb)：`let + object + Adv` (e.g., out) *![](screenshots/screenshot_3d7166beb-d0e3-4b9c-81af-ebbe0ca1d32a.jpg) (展示 let me out)
    *   过去分词 (Past Participle)：`have/get + object + V-ed` (e.g., washed, done) *![](screenshots/screenshot_479927537-5e57-40b3-a3ec-dd825e64ce5e.jpg) (展示 have my car washed)

### 核心使役动词详解
#### 1.  Make [原片 @ 00:26](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=26)
*   **隐含意义**：强制、迫使、导致（常带强迫性）。
*   **宾语补足语形式**：
    *   `make + object + V原形` (裸不定式)：`I make the wolf eat carrots.` (我迫使狼吃胡萝卜) *![](screenshots/screenshot_56721b382-c7cd-4fed-b278-5d61663393bf.jpg)
    *   `make + object + Adj`：`The carrot made the wolf full.` (胡萝卜让狼饱了) *![](screenshots/screenshot_6d214f386-2cd2-4700-bfeb-b42c0f284a52.jpg)
*   **关键点**：宾语补足语不可省略，否则句意不完整 (e.g., `I make the wolf` ❌)。

#### 2.  Let [原片 @ 03:15](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=195)
*   **隐含意义**：允许、让（某人做某事）。
*   **宾语补足语形式**：
    *   `let + object + V原形` (裸不定式)：`I let the wolf eat some chocolate.` (我允许狼吃点巧克力) *![](screenshots/screenshot_7568ec33d-829b-4f38-aa0c-ba0751532361.jpg)
    *   `let + object + Adv`：`Let me out!` (让我出去！) *![](screenshots/screenshot_84b35d2f6-16c6-4636-ac99-9ad1845aa92a.jpg)
*   **关键点**：表示“允许”，非强制。

#### 3.  Have [原片 @ 03:53](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=233)
*   **隐含意义**：安排、指派（任务或责任），委托（服务）。
*   **宾语补足语形式**：
    *   `have + object (人) + V原形` (指派具体人做事)：`I have the wolf wash the dishes.` (我让狼洗盘子 - 明确指派狼) *![](screenshots/screenshot_97223266f-5d61-4ecc-806c-86ebce2ee6e2.jpg)
    *   `have + object (物/事) + V-ed` (过去分词) (委托服务/遭遇事件)：
        *   委托服务：`I have my car washed.` (我让人洗了我的车 - 不指定谁洗) / `I have my nails done.` (我让人修了指甲) *![](screenshots/screenshot_1033dcef67-e39a-4379-a089-e80568349bb5.jpg)
        *   遭遇不幸（非主动安排）：`I had my wallet stolen yesterday.` (我的钱包昨天被偷了) *![](screenshots/screenshot_11f557073a-b0e5-401c-9e47-6d48fb0fa02b.jpg)
*   **关键点**：`have + object + V-ed` 结构隐含被动含义 (e.g., `My car was washed by someone`)。

#### 4.  Get [原片 @ 06:10](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=370)
*   **隐含意义**：说服、劝说、促使（某人做某事）。
*   **宾语补足语形式**：`get + object (人) + to V` (带 to 的不定式 Full Infinitive)。
    *   `I'll get the wolf to clean the kitchen.` (我会让/说服狼清理厨房) *![](screenshots/screenshot_121b35199a-0ce8-4026-9d9f-37b37d722737.jpg)
*   **关键点**：必须使用 `to + V原形`，**不可**用裸不定式 (❌ `get the wolf clean`)。*![](screenshots/screenshot_13b15e5b7f-2bd4-4b15-b0b9-270a34da8222.jpg) (强调带 to 的不定式)
*   **特殊用法**：`get + object (物) + V-ed` (委托/遭遇，同 `have`)：`I'll get my nails done.` (我让人修指甲)

### 使役动词对比与总结 [原片 @ 07:49](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=469)
*   **共同点**：均需接宾语和宾语补足语。
*   **核心差异总结**：
    | 使役动词 | 隐含意义           | 典型宾语补足语形式          | 关键语法点                 |
    | :------- | :----------------- | :-------------------------- | :------------------------- |
    | **Make** | 强迫、导致         | `+ object + V原形` / `+ Adj` | 最强制                     |
    | **Let**  | 允许               | `+ object + V原形` / `+ Adv` | 最许可                     |
    | **Have** | 安排、指派、委托   | `+ object (人) + V原形`      | 明确责任人                 |
    |          |                    | `+ object (物) + V-ed`       | 委托服务或遭遇（被动含义） |
    | **Get**  | 说服、劝说、促使   | `+ object (人) + to V`       | **必须带 to**              |
    |          |                    | `+ object (物) + V-ed`       | 同 `have` (委托/遭遇)      |
*   **宾语补足语形式是核心区别**：
    *   Make, Let, Have (指派人)：接 **动词原形** (裸不定式)。
    *   Get (指派人)：接 **带 to 的不定式** (`to + V原形`)。
    *   Have/Get (委托事/物 或 遭遇)：接 **过去分词** (`V-ed`)。
*   **其他使役动词**：理解 `make, let, have, get` 后，类似动词 (如 `force`, `allow`, `drive`, `leave`) 的用法更容易掌握。[原片 @ 08:45](https://www.bilibili.com/video/BV1XY411J7aG_p10?t=525)

## AI 总结
本课系统讲解了英语核心使役动词 **Make, Let, Have, Get**：
1.  **核心结构**：必须带宾语及宾语补足语 (V原形、Adj、Adv、to V、V-ed)，否则句意不完整。
2.  **关键差异**：
    *   **Make** (强迫/导致)：+ V原形/Adj (`make sb do/adj`)。
    *   **Let** (允许)：+ V原形/Adv (`let sb do/out`)。
    *   **Have** (安排/委托)：
        *   明确人：+ V原形 (`have sb do`)。
        *   委托/遭遇：+ V-ed (`have sth done`)。
    *   **Get** (说服)：
        *   说服人：**必须** + to V (`get sb to do`)。
        *   委托/遭遇：+ V-ed (`get sth done`，同 `have`)。
3.  **重点注意**：`Get` 后接人时，宾语补足语**必须用带 `to` 的不定式** (`to do`)，区别于其他三个动词的裸不定式 (`do`)。
4.  **实用场景**：掌握四种动词的隐含意义和结构差异，是准确表达“使让”含义的关键。
